# Interactive cis-QTL browser

<div class="ref-description section level2">

Launches a Shiny application for exploring cis-QTL associations. Users
can select genes, view associated SNPs, and visualize the genotype
effect on phenotypes using `plotGenotypeEffect()`.

</div>

<div class="section level2">

## Usage

<div class="sourceCode">

``` r
run_cis_qtl_browser(
  tqe,
  cis_gr,
  gene_col = "gene_symbol",
  snp_col = "variant_id",
  phenotype_col = "phenotype_id",
  pval_col = "pval_nominal",
  pval_thresh = NULL
)
```

</div>

</div>

<div class="section level2">

## Arguments

-   tqe:

    A
    [tQTLExperiment](https://github.com/vjcitn/tQTLExperiment/reference/tQTLExperiment.md)
    with genotype data.

-   cis\_gr:

    A
    [GenomicRanges::GRanges](https://rdrr.io/pkg/GenomicRanges/man/GRanges-class.html)
    object containing cis-QTL results (e.g., from `readTQTL()`).

-   gene\_col:

    Name of the metadata column in `cis_gr` containing gene symbols.
    Default: `"gene_symbol"`.

-   snp\_col:

    Name of the metadata column in `cis_gr` containing SNP IDs. Default:
    `"variant_id"`.

-   phenotype\_col:

    Name of the metadata column in `cis_gr` containing phenotype/gene
    IDs. Default: `"phenotype_id"`.

-   pval\_col:

    Name of the metadata column in `cis_gr` containing p-values.
    Default: `"pval_nominal"`.

-   pval\_thresh:

    Optional numeric threshold for filtering associations by p-value. If
    `NULL` (default), all associations are included.

</div>

<div class="section level2">

## Value

A Shiny application object (invisibly). Call this function in an
interactive R session to launch the browser.

</div>

<div class="section level2">

## Details

The browser provides two tabs:

-   **Genotype effect**: Beeswarm plot of phenotype values by SNP
    genotype, generated by `plotGenotypeEffect()`.

-   **cis associations**: Filterable data table of all associations for
    the selected gene.

Navigation is via dropdown menus: select a gene symbol (searchable),
then a cis-SNP for that gene, then click "Plot" to visualize.

</div>

<div class="section level2">

## Examples

<div class="sourceCode">

``` r
if (interactive()) {
  # Load MAGE chr17 genotypes and pre-computed tensorQTL results
  plink_paths <- cache_mage_chr17_plink(verbose = FALSE)
  plpre <- tools::file_path_sans_ext(plink_paths[["bed"]])
  data(mageSEfilt, package="CSHLvc2026")
  cd <- as.data.frame(colData(mageSEfilt))
  mm <- model.matrix(~ factor(batch) + population + sex, data = cd)
  mm <- mm[, -1, drop = FALSE]   # remove (Intercept)
  tqe <- tQTLExperimentFromRSE(
   se              = mageSEfilt,
   plinkPrefix     = plpre,
   covariateMatrix = mm,
   genome          = "hg38"
   )
  tqe <- addGeneSymbols(tqe)
  demd = cache_demo_data()
  res <- readTQTL(dirname(demd$parquet), mode = "cis_nominal", x = tqe)
  newpairs = res$pairs
  newpairs = annotateTensorQTLResults(newpairs)

  # Launch browser with p-value filter
  run_cis_qtl_browser(tqe, newpairs, pval_thresh = 0.05)
}
```
